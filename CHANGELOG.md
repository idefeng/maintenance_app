- 原生维护工具的磁盘清理与登录项页面新增搜索、筛选和单项详情查看

- 原生维护工具新增 App 内日志预览，可直接查看定时任务最近日志
- 原生维护工具新增报告摘要复制与 Markdown 摘要导出
- 原生维护工具按 Figma 设计稿调整窗口布局、标题区操作、指标排版、定时任务行和日志预览弹窗
- 原生维护工具新增磁盘空间环形图、文件整理额外路径配置、登录项应用图标和定时任务行图标
- 原生维护工具新增总览健康检查，集中提示低磁盘空间、过期报告、清理失败、登录项残留、待处理目录、无效整理路径、缺失定时任务和过期任务产物
- 原生维护工具将只读预览入口升级为“扫描”功能，明确扫描不会删除文件，仅生成报告并刷新健康检查
- 文件整理工具新增 `runtime/config/source-rules.json` 额外来源配置，统一脚本和单独整理脚本都会读取
- 为应用残留清理工具补充 `scripts/install_launch_agent.sh`，供 App 重新安装定时任务
- README 新增原生维护工具构建、运行和功能说明
